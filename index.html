Tentang

Website ini menggunakan metode CRISP-DM untuk proses klasifikasi, meliputi langkah-langkah berikut:

  • Business Understanding: Menentukan tujuan dan masalah.
  • Data Understanding: Memahami struktur dan isi dataset.
  • Data Preparation: Menyiapkan data untuk model.
  • Modeling: Membangun model menggunakan algoritma KNN.
  • Evaluation: Mengevaluasi performa model.
  • Deployment: Mengimplementasikan model dalam aplikasi.
